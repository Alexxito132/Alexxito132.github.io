Explicando las etiquetas HTML

Bienvenido a HTML

En esta página vas a encontrar las etiquetas más esenciales de HTML explicadas de forma clara y sencilla. La idea es que puedas reconocerlas, entender para qué sirven, conocer sus atributos principales y ver ejemplos para que sepas cómo usarlas. Todo pensado para que al terminar, tengas la base necesaria para empezar a armar tus propias páginas sin perderte entre tanta etiqueta.

Hablemos de <!DOCTYPE html>:

No es una etiqueta HTML como <p> o <h1> sino una declaración que va en la primera linea de cualquier documento HTML. Su función es decirle al navegador que el documento está en HTML5 con el fin de tome las reglas modernas del estándar.
Si esta no se coloca el navegador entra en modo quirks que intenta preestablecer el diseño antiguo de las páginas pero esto puede afectar al diseño y hacer que algunos estilos no se apliquen.

Etiqueta HTML

La etiqueta <html> es el elemento raíz de cualquier documento HTML. Contiene todo el contenido de la página, incluyendo el encabezado <head> y el cuerpo <body>. Sirve para indicar al navegador que todo lo que está dentro es código HTML.

Etiqueta HEAD

Hablemos de la <head> este es la cabecera del documento HTML, pero no es el encabezado visual que se ve en la pagina ese se modifica a travez de <header>. Su función es contener metadatos que es información sobre la página que no se muestra a los usuarios, pero que los navegadores usan para mostrar el sitio correctamente.
Puede usar atributos globales como id, class, style, lang, etc.

Etiqueta BODY

Es aquella etiqueta que define el cuerpo del documento HTML. Contiene todo el contenido visible: texto, imágenes, videos, enlaces, listas, tablas, formularios, etc.
Puede usar atributos globales como id, class, style, lang, etc.

Etiquetas H1 hasta H6

Las etiquetas <h1> a <h6> definen niveles de encabezado, del más importante (<h1>) al menos (<h6>). Se usan para organizar el contenido y crear una jerarquía visual y semántica.
Tienen atributos como id, class, style, lang, title, etc.

Ejemplo:

<h1>HTML es el lenguaje que estructura las páginas web...</h1>

<h2>HTML es el lenguaje que estructura las páginas web...</h2>

<h3>HTML es el lenguaje que estructura las páginas web...</h3>

<h4>HTML es el lenguaje que estructura las páginas web...</h4>

<h5>HTML es el lenguaje que estructura las páginas web...</h5>
<h6>HTML es el lenguaje que estructura las páginas web...</h6>

Etiqueta <p>

La etiqueta <p> llamada asi por paragraph. Se usa para agrupar y separar bloques de texto en una página web como párrafos. De igual manera no tiene atributos propios pero usa los globales.

Ejemplo:

<p>HTML es el lenguaje que estructura las páginas web...<p1>

Etiqueta <a>

Esta etiqueta permite crear enlaces que pueden llevar a otra página web, apuntar un archivo para descargar, saltar a otra parte de la misma página o abrir el correo electrónico del usuario.

Etiqueta <img>

Esta etiqueta es muy importante pues es la que permite insertar imágenes en el contenido HTML. Es una etiqueta vacía y necesita atributos para funcionar, se usa cuando la imagen forma parte del contenido. Etiquetas OL y LI

La etiqueta <ol> define una lista ordenada, donde cada elemento se numera automáticamente. Dentro de ella, la etiqueta <li> representa cada elemento de la lista. Se usan para mostrar pasos, rankings o cualquier contenido donde el orden sea importante.
<ol> puede tener atributos como type, start y reversed, mientras que <li> puede usar value para definir un número específico, además de atributos globales como id, class, style, lang, title, etc.

Ejemplo:

<ol>

  <li>Precalentar el horno a 180°C.</li>

  <li>Mezclar harina, azúcar y huevos.</li>

  <li>Hornear durante 30 minutos.</li>

</ol>

Etiqueta UL

La etiqueta <ul> define una lista no ordenada, donde cada elemento se representa con una viñeta o marcador en lugar de un número. Dentro de ella, la etiqueta <li> representa cada elemento de la lista. Se usa cuando el orden de los elementos no es importante.
<ul> puede usar atributos globales como id, class, style, lang, title, etc., mientras que <li> también puede usar estos atributos y, en listas ordenadas, el atributo value para definir un número específico.

Ejemplo:

<ul>

  <li>Manzana</li>

  <li>Pera</li>

  <li>Banana</li>

</ul>
